### General Requirements

What You Must Submit:
1. Source Code include:
- Well-commented Python files

2. README.md File include: 
- Project title and your name
- Brief description of what your project does
- How to run your program (step-by-step instructions)
- List of features implemented
- Any dependencies or installation requirements
- Sample usage/screenshots (optional but encouraged)

3. Complexity Analysis Document include:
- Data structures used: Explain which ones you used and why you chose them
- Time complexity analysis: Big-O notation for all major operations
- Space complexity analysis: Memory usage for key data structures
- Algorithm comparison: Compare at least 2 different approaches for a key operation
- Trade-offs: Discuss design decisions and why you made them
